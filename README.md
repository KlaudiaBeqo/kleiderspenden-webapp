# Kleiderspenden Web App

## Beschreibung
Diese Webanwendung ermöglicht es Benutzern, Kleiderspenden zu registrieren.

## Funktionen
- Eingabe von Name, E-Mail und Kleidungsstück
- Validierung der Eingaben
- Anzeige einer Bestätigungsmeldung
- Anzeige der Spenden in einer Liste

## Technologien
- HTML
- CSS
- JavaScript

## Nutzung
1. Formular ausfüllen
2. Auf "Spenden" klicken
3. Spende wird angezeigt
